# 🛍️ PAYUEE e-Shop Interactive E-commerce Dashboard

## 📘 Project Overview
The **PAYUEE e-Shop Dashboard** is an **interactive Power BI report** built to analyze and visualize key metrics of  payuee platform.  
It provides a comprehensive view of **sales performance**, **customer insights**, and **geographical distribution**, helping business owners and analysts make **data-driven decisions**.

---

## 🚀 How to Use
1. Download the Excel & Power bi file from this repo.  
2. Open the **DASHBOARD** sheet to interact with the report.

## 🚀 Key Features
- **Revenue Overview:** Displays total revenue, quantity sold, and average unit price with year-over-year (YoY) comparisons.  
- **Monthly Sales Trend:** Line chart visualizing monthly growth patterns in sales.  
- **Customer Insights:** Highlights the top 5 customers with a donut chart representation.  
- **Product Insights:** Ranks top 5 units by quantity sold.  
- **Geographical Sales Distribution:** Interactive map showing top-performing countries by sales value.  
- **Dynamic Filters:** Allows easy filtering and exploration of data by various dimensions.

---

## 🧩 Data Model
The dashboard is powered by a **star schema model** with the following tables:

- `fact_table` – main transactional data (sales, revenue, quantity)  
- `customer_dim` – customer details  
- `item_dim` – product or item information  
- `store_dim` – store location and type  
- `time_dim` – time-based analysis for trends  
- `trans_dim` – transaction metadata
